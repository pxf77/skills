# Fantasy Web-Novel Trope Library

Use this reference to identify popular fantasy web-fiction conflict structures and convert them into visual image-drama beats.

## Broken Engagement Reversal

- **Conflict:** A powerful family or sect publicly humiliates the protagonist by breaking an engagement.
- **Appeal:** The protagonist stays calm, reveals hidden power or identity, and reverses the social pressure.
- **Iconic visuals:** Ancestral hall, jade engagement token cracking, crowd in sect robes, antagonist sneer, protagonist's spiritual aura igniting.
- **Rhythm:** Oppression -> humiliation -> quiet reaction -> hidden power clue -> public reversal -> stunned crowd.

## Underdog Awakening

- **Conflict:** A mocked weak character is pushed to the edge.
- **Appeal:** Dormant bloodline, divine bone, ancient soul, or forbidden artifact awakens.
- **Iconic visuals:** shattered training stone, golden or black aura, ancient runes on skin, sky tearing open, shocked elders.
- **Rhythm:** Failure -> ridicule -> crisis -> inner spark -> power manifestation -> new status.

## Rebirth Return

- **Conflict:** A betrayed character returns to an earlier moment with future knowledge.
- **Appeal:** Calm control, revenge setup, emotional restraint, second-chance domination.
- **Iconic visuals:** blood moon, broken mirror, palace corridor, old enemy reappearing, protagonist's colder gaze.
- **Rhythm:** Death memory -> awakening -> recognition of old scene -> controlled decision -> first counter-move.

## Hidden Identity Reveal

- **Conflict:** Others misjudge the protagonist's status.
- **Appeal:** The concealed emperor, war god, saint, demon lord, or heir reveals a symbol no one can deny.
- **Iconic visuals:** imperial token, divine seal, black-gold robe, kneeling crowd, aura pressure bending the environment.
- **Rhythm:** Insult -> warning ignored -> token reveal -> atmosphere collapse -> public kneeling.

## Empress Return

- **Conflict:** A fallen or reborn female ruler returns to court, sect, or battlefield.
- **Appeal:** Regal pressure, revenge, elegance, emotional distance.
- **Iconic visuals:** phoenix crown, red-gold robes, palace steps, snow or petals, kneeling ministers, phoenix fire.
- **Rhythm:** Entrance -> recognition -> political pressure -> single decisive line -> power display.

## Demon Lord Pressure Scene

- **Conflict:** A feared demon lord or dark sovereign confronts an enemy, disciple, or old lover.
- **Appeal:** Overwhelming aura, danger, control, forbidden tenderness or threat.
- **Iconic visuals:** black throne, crimson moon, demonic mist, chains, dark armor, red eyes, shattered hall.
- **Rhythm:** Silent arrival -> pressure spreads -> enemy trembles -> close-up threat -> magic suppression.

## Master-Disciple Confrontation

- **Conflict:** A disciple challenges a master, or a master hides sacrifice and severity.
- **Appeal:** Emotional restraint, hierarchy, betrayal ambiguity, dramatic close-ups.
- **Iconic visuals:** cold mountain peak, white robes, sword light, kneeling disciple, snow, broken jade pendant.
- **Rhythm:** Formal distance -> accusation -> memory symbol -> sword or spell raised -> unresolved emotional cliffhanger.

## Sect Trial

- **Conflict:** The protagonist is judged by elders or peers.
- **Appeal:** Public pressure, unfair rules, sudden proof of talent.
- **Iconic visuals:** trial platform, elder seats, glowing array, sect disciples, spiritual root test, ancient bell.
- **Rhythm:** Accusation -> evidence against protagonist -> test begins -> impossible result -> elders stand in shock.

## Secret Realm Treasure Fight

- **Conflict:** Multiple factions fight for an artifact in a dangerous realm.
- **Appeal:** survival, betrayal, artifact recognition, sudden power leap.
- **Iconic visuals:** floating ruins, glowing treasure, beasts, broken bridge, storm clouds, competing cultivators.
- **Rhythm:** discovery -> scramble -> trap -> betrayal -> artifact chooses protagonist -> escape cliffhanger.

## Heavenly Tribulation Manifestation

- **Conflict:** A breakthrough invites impossible lightning tribulation.
- **Appeal:** scale, awe, public disbelief, character transcendence.
- **Iconic visuals:** purple thunder clouds, lightning dragons, cracked earth, robes whipping in wind, golden core or divine mark.
- **Rhythm:** sky darkens -> crowd panic -> protagonist steps forward -> lightning strike -> figure remains standing.

## Divine-Demon Battlefield

- **Conflict:** Armies, gods, demons, or ancient clans collide.
- **Appeal:** epic scale, sacrifice, faction identity, overwhelming stakes.
- **Iconic visuals:** burning sky, giant weapons, banners, demonic tide, celestial formation, wounded hero standing alone.
- **Rhythm:** battlefield reveal -> faction clash -> hero isolated -> power surge -> final wide shot cliffhanger.
